# User Flow

## Registration

User opens app
↓
User creates account
↓
User logs in

## Join Lecture

User enters lecture code
↓
User joins lecture
↓
User sees lecture board

## Create Lecture

Lecturer creates lecture
↓
System generates code
↓
Students join using code
↓
Lecture starts

## Permissions

Lecturer:
- Create lecture
- Start lecture
- End lecture

Student:
- Join lecture using code
- View lecture content
